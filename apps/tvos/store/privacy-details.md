# App Privacy answers

Use these answers for the tvOS app unless the production logging configuration changes.

- Tracking: No.
- Data linked to the user: No.
- Accounts or persistent user identifiers: None.
- Advertising: None.
- Data type: Other Data — technical network data such as IP address and request logs.
- Purpose: App Functionality and Security.
- Collection status: Declare as collected if Cloudflare, R2, Coolify, or another provider retains these logs beyond the immediate request.

Do not choose “Data Not Collected” until retention is confirmed for every hosting and CDN provider. The privacy text intentionally describes the conservative case.
