## [Unreleased]

- Jest testing setup
- CODEOWNERS file

## [0.1.0] - 2024-XX-XX

### Added
- Initial release of Lofiever
- 24/7 lofi music streaming
- Real-time synchronized listening experience
- Live chat with AI moderation
- Track queue visualization
- Multi-language support (English, Portuguese)
- Responsive design with dark mode
- Docker deployment support
- PostgreSQL database with Prisma ORM
- Redis for real-time features
- Socket.IO for WebSocket communication
- Icecast/Liquidsoap streaming infrastructure

### Technical Stack
- Next.js 15 with App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Prisma ORM
- PostgreSQL
- Redis
- Socket.IO
- Docker & Docker Compose
